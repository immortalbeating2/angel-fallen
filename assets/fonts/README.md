# Fonts

本目录用于存放项目正式字体资源。

目标：

- 为 UI、标题、章节标题、档案页和提示信息提供统一字体体系
- 区分正文可读性字体与标题装饰字体

格式要求：

- 最终入库格式：`.ttf` 或 `.otf`
- 优先保证中英文都可用
- 正文字体优先高可读性，不追求过度装饰

建议分工：

- 正文字体：菜单、HUD、面板正文、档案说明
- 标题字体：主菜单标题、章节标题、结算标题、结局标题

命名示例：

- `font_ui_body_main.ttf`
- `font_ui_title_relic.otf`
- `font_archive_body_cn.ttf`

收稿标准：

- 必须验证中文常用字覆盖
- 必须验证英文、数字、标点完整
- 标题字体允许更强风格，但不能明显破坏阅读
