# Enemies

本目录用于存放敌人相关精灵资源。

当前阶段建议优先放入：

- 敌人族群风格板
- 普通敌 / 精英敌 / Boss 轮廓测试图
- 章节敌群方向稿

当前阶段不建议直接定稿：

- 最终运行时敌人 spritesheet
- Boss 多阶段最终演出帧
- 与真实攻击 telegraph 完全绑定的终稿动画

格式要求：

- 概念与方向稿：`.png`
- 最终运行时精灵：`.png`

命名示例：

- `enemy_chapter_1_family_board_v1.png`
- `enemy_slime_topdown_concept_v1.png`
- `enemy_skeleton_attack_sheet_v1.png`
