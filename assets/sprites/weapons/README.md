# Weapons

本目录用于存放武器、本体弹幕、命中特效关联资源。

当前阶段建议优先放入：

- 武器方向稿
- 投射物方向稿
- 进化前后视觉差异对比稿
- 武器表现规划参考图

当前阶段不建议直接定稿：

- 最终攻击序列帧
- 与真实伤害节奏绑定的弹幕终稿
- 高密度战斗特效终稿

格式要求：

- 概念与方向稿：`.png`
- 最终运行时武器与弹幕精灵：`.png`

命名示例：

- `wpn_holy_cross_concept_v1.png`
- `wpn_magic_missile_projectile_v1.png`
- `wpn_archive_relic_set_v1.png`
