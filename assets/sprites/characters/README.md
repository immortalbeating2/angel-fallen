# Characters

本目录用于存放角色相关精灵资源。

当前阶段建议优先放入：

- 俯视角角色方向稿
- 职业 silhouette 测试图
- 角色概念图
- 动画拆分规划参考图

当前阶段不建议直接定稿：

- 最终运行时 spritesheet
- 与真实攻击节奏完全绑定的战斗动画终稿

格式要求：

- 概念与方向稿：`.png`
- 最终运行时精灵：`.png`
- 后续若进入 Godot 动画使用，需再整理为 spritesheet 或 SpriteFrames 兼容资源

命名示例：

- `char_knight_topdown_concept_v1.png`
- `char_mage_silhouette_test_v1.png`
- `char_rogue_idle_sheet_v1.png`
