# UI Cards

本目录用于存放奖励卡片、档案卡片、记录卡片等卡片式 UI 资源。

建议内容：

- Run Result 奖励卡
- 挑战层奖励卡
- 档案记录卡
- 结局摘要卡

格式要求：

- 透明背景 `.png`
- 预留图标区、标题区、明细区
- 支持后续多稀有度或状态变体

命名示例：

- `ui_card_reward_archive_v1.png`
- `ui_card_challenge_reward_v1.png`

相关提示词包：

- `spec-design/stage7-gemini-copyable-prompt-pack-2026-03-24.md`
