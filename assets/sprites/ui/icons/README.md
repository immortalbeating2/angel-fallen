# UI Icons

本目录用于存放系统图标、路线图标、结局图标、隐藏层/挑战层图标等。

格式要求：

- 最终运行时使用透明背景 `.png`
- 优先保证缩小后仍清晰
- 先做单色/双色高对比版本，再决定是否扩展彩色版

建议题材：

- `redeem` / `fall` / `balance`
- `route`
- `memory shard`
- `archive seal`
- `hidden layer`
- `challenge badge`
- `forge ember`
- `void sigil`

命名示例：

- `ui_icon_memory_shard_v1.png`
- `ui_icon_archive_seal_v1.png`
- `ui_icon_route_outcomes_sheet_v1.png`
- `ui_icon_archive_systems_sheet_v1.png`
