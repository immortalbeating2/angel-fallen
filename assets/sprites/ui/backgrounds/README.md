# UI Backgrounds

本目录用于存放菜单、营地、记忆祭坛、档案室、挑战档案等 UI 背景图。

建议内容：

- 主菜单背景
- 安全营地背景
- 记忆祭坛背景
- Ending Archive / Hidden Layer Track / Challenge Archive 背景

格式要求：

- 最终运行时使用 `.png`
- 需要透明分层时可拆多张 `.png`
- 画面需预留 UI 覆盖区域，避免信息堆在中心

命名示例：

- `ui_bg_main_menu_ruined_halo_v1.png`
- `ui_bg_safe_camp_last_fire_v1.png`
- `ui_bg_memory_altar_echoes_v1.png`
- `ui_bg_challenge_archive_v1.png`
