# Assets

本目录是《堕落天使》Stage 7 正式资产总入口。

当前主要子目录：

- `audio/`：BGM、环境音、SFX
- `fonts/`：项目字体资源
- `shaders/`：通用表现层 Shader
- `sprites/`：UI、地表、特效、角色、敌人、武器等图形资源

当前使用原则：

- `audio`、`fonts`、`shaders`、`sprites/ui`、`sprites/effects` 可较早推进到正式收稿。
- `sprites/characters`、`sprites/enemies`、`sprites/weapons` 当前优先存放方向稿、规划稿、silhouette 测试，不建议过早当作最终运行时精灵终稿。
